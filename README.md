# Internship Portfolio

Personal portfolio site for Phat Pham, Software Engineering (Honours) student at Flinders University seeking a 20-week industry placement (available from September 2026).

Live projects, skills, and contact details, with an animated 3D hero (React Three Fiber desk scene plus a particle-network background).

## Stack

- React + Vite + TypeScript
- Tailwind CSS v4
- React Three Fiber / drei / Three.js (3D desk scene, particle background)
- Motion (scroll and UI animation)

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

See `docs/superpowers/specs/2026-07-20-portfolio-design.md` for the full design spec.
